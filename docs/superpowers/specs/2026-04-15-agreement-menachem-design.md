### 1.1 אוטומציית ביטול מנוי

**מצב היום:**
מנוי רוצה לבטל → כותב למנחם → מנחם נכנס ל-Cardcom, מבטל הוראת קבע → נכנס ל-Skolar, מוריד גישה → נכנס ל-WhatsApp, מוציא מקבוצה. 3 מערכות, 10-15 דקות לביטול. לפעמים שוכח שלב אחד → לקוח ממשיך לשלם → כועס → נזק למוניטין.

**מה ישתנה:**
לקוח כותב "ביטול" בוואטסאפ → המערכת מבטלת ב-3 מערכות תוך 30 שניות → שולחת הודעת פרידה חמה → מתזמנת win-back אוטומטי (30/60/90 יום).

**שווה למנחם:**
- 0 דקות עבודה על ביטולים
- 0 סיכוי לטעות (לקוח שמשלם ולא מקבל / לקוח שלא משלם ומקבל)
- Win-back אוטומטי שמחזיר ~10% מהמבטלים = מנויים שנשמרים

**מספרים:**
- היום: ~5-15 דקות × מספר ביטולים/חודש = שעות של מנחם
- מחר: 0 דקות. מנחם לא יודע שמישהו ביטל עד שהוא מסתכל בדשבורד

**זמן ביצוע:** 1-2 שבועות
**תמחור:** 3,500₪

### 1.2 Dunning — הצלת כרטיסי אשראי שנכשלים

**מצב היום:**
כרטיס אשראי נכשל (פג תוקף, חסימה, מסגרת) → מנחם לא יודע → המנוי "נופל" בשקט → הכנסה אבודה. או: מנחם שם לב → כותב ללקוח ידנית → מבזבז זמן.

**מה ישתנה:**
כרטיס נכשל → ניסיון חוזר אחרי 3 ימים → ניסיון שלישי אחרי שבוע → הודעת WhatsApp אוטומטית ללקוח: "היי, הכרטיס לא עבר. עדכן כאן 👇" → אם לא עודכן תוך 14 יום → ביטול אוטומטי.

**שווה למנחם:**
כל מנוי שנשמר = **540₪/שנה** (45₪ × 12). הנתונים מראים:
- ~2% כשלי אשראי חודשיים = ~17 כרטיסים
- Dunning אוטומטי מציל 50-70% = **8-12 מנויים/חודש**
- **= 360-540₪ הכנסה חודשית שנשמרת**
- **= 4,320-6,480₪/שנה**

**מספרים:**

| בלי dunning | עם dunning |
|-------------|------------|
| 17 כשלים → 17 מנויים אבודים | 17 כשלים → 5-7 אבודים, 10-12 נשמרים |
| הפסד: 9,180₪/שנה | הפסד: 2,700-3,780₪/שנה |
| | **חיסכון: 5,400-6,480₪/שנה** |

**זמן ביצוע:** 1-2 שבועות
**תמחור:** 3,000₪
**החזר השקעה:** 5-6 חודשים. משם — רווח נקי.

### 2.1 Referral אוטומטי — "כל מנוי = משווק"

**מצב היום:**
מנחם מבקש לשתף בסטטוס. חלק עושים, חלק לא. אין מעקב. אין תגמול. אין מערכת.

**מה ישתנה:**
כל מנוי מקבל לינק אישי. משתף עם חבר → חבר נרשם → המנוי מקבל הודעה: "בזכותך הצטרף/ה [שם]! קיבלת חודש מתנה 🎁" → אוטומטי.

**שווה למנחם:**

| מדד | חישוב |
|-----|-------|
| מנויים פעילים | 836 |
| % שישתפו (שמרני) | 10% = 84 שיתופים/חודש |
| % שיירשמו מהשיתוף | 20% = ~17 מנויים חדשים/חודש |
| הכנסה מכל מנוי חדש | 5₪ ראשון + 45₪ × 11 = 500₪/שנה |
| **הכנסה שנתית נוספת** | **17 × 500₪ = 8,500₪/שנה** |
| עלות פרסום | **0₪** |

וזה מצטבר. חודש 1: +17. חודש 2: +19 (כי יש יותר מנויים שמשתפים). חודש 6: +25. אפקט שלג.

**הפיץ' למנחם:**
> "היום אתה מבקש מאנשים לשתף ומקווה לטוב. מחר — כל מנוי מקבל לינק אישי, משתף, והחבר שלו מצטרף. המנוי מקבל חודש מתנה אוטומטית. אתה לא עושה כלום. זה עובד 24/7."

**זמן ביצוע:** 2-3 שבועות
**תמחור:** 4,000₪
**החזר השקעה:** 6 חודשים. משם — מכונה שעובדת לבד.

### 2.2 בוט WhatsApp — משפך מכירה 24/7

**מצב היום:**
הבוט הנוכחי נותן סיפורים חינמיים ומופנה לקבוצות. אין משפך מכירה אוטומטי.

**מה ישתנה:**

```
אדם חדש שולח הודעה לבוט
         ↓
"היי! רוצה לשמוע סיפור מדהים בקולו של מנחם שרון? 🎧"
         ↓
שולח סיפור טעימה (2 דקות)
         ↓
"אהבת? יש עוד 190+ סיפורים. הצטרף למועדון — 5₪ בלבד לחודש ראשון"
         ↓  [כן!]                    ↓  [לא עכשיו]
לינק סליקה          "בסדר! שולח לך סיפור נוסף מחר. בינתיים, שתף חבר 👇"
         ↓                           ↓
מנוי חדש!                   drip: סיפור ביום 3, ביום 7 → "מוכן?" → סליקה
```

**שווה למנחם:**
- מוכר 24/7 בלי שמנחם או יוסף עושים משהו
- כל אדם שנכנס לבוט = ליד חם שמקבל טיפול אוטומטי
- נתונים: 160 כניסות לבוט (פברואר), 450 כניסות (מרץ) — המספרים כבר גדלים

**מספרים (שמרני):**

| מדד | חישוב |
|-----|-------|
| כניסות לבוט/חודש | ~300 (ממוצע) |
| המרה לסיפור טעימה | 80% = 240 |
| המרה למנוי (מיידי) | 10% = 24 מנויים |
| המרה מ-drip (30 יום) | 5% = 12 מנויים |
| **סה"כ חדשים/חודש** | **~36** |
| **הכנסה שנתית נוספת** | **36 × 500₪ = 18,000₪/שנה** |

**זמן ביצוע:** 2-3 שבועות

### 2.3 קמפיין כרטיס טיסה — "בום חד-פעמי"

**מצב היום:**
מנחם רוצה לעשות את זה. יש רעיון, אין מערכת.

**מה נבנה:**
- מנגנון הגרלה אוטומטי: שיתוף בסטטוס/קבוצה → כניסה לבוט → מספר הגרלה
- מיני-הגרלות שבועיות (פליימוביל, ארוחה) → שומרים מומנטום
- גרנד פרייז: כרטיס טיסה זוגי לאירופה (ל"ג בעומר)
- כל שיתוף = כניסה לבוט = ליד חדש

**שווה למנחם:**

| מדד | חישוב |
|-----|-------|
| חשיפה | 17,000 חברי קהילה × שיתוף = עשרות אלפי חשיפות |
| כניסות לבוט צפויות | 1,000-2,000 |
| המרה למנויים | 10-15% = **100-300 מנויים חדשים** |
| הכנסה מהקמפיין | 100 × 500₪ = **50,000₪/שנה (מינימום)** |
| עלות כרטיס טיסה | ~3,000-5,000₪ |

**הפיץ' למנחם:**

**מספרים — סימולציה:**

### 3.1 קול מנחם ב-11Labs — "100 סיפורים בלי אולפן"

**מצב היום:**
מנחם נכנס לאולפן → כותב תסריט → מקליט → עורך → מעלה. סיפור אחד = שעה-שעתיים עבודה. מוגבל ל-~30 סיפורים/חודש.

**מה ישתנה:**
מנחם מקליט 30 דקות training material (פעם אחת!) → 11Labs יוצר clone של קולו → מנחם כותב תסריט → AI מקריא בקולו → מנחם מאשר (30 שניות) → עולה למאגר.

| מדד | היום | עם 11Labs |
|-----|------|-----------|
| זמן ליצירת סיפור | 60-120 דק | 10-15 דק (כתיבה + אישור) |
| סיפורים/חודש | ~30 | 100-300 |
| מנחם באולפן | כל יום | כשבא לו |
