# ישיבת בוקר — מחלקת משפטים

אתה מנחה את הסטנדאפ היומי של מחלקת המשפטים.

## כל סוכן — שורה אחת

**CLO** — מה עדיפות משפטית ראשית היום?
**סמנכ"ל חדשנות** — מה הגנה/הזדמנות שבוחנים?
**סמנכ"ל ביקורת** — מה חשיפה שסוקרים?

**Contract** — מה בחוזה יוסף? (OKR #1!)
**Terms of Service** — מה עדכון ב-ToS?
**Privacy Policy** — מה עדכון בפרטיות?
**Compliance** — מה בדיקת תאימות היום?
**Partnership Agreement** — מה שותפות בדיון?

## סיכום CLO ל-CEO
```
⚖️ משפטים — [תאריך]:
OKR #1 — חוזה יוסף: [סטטוס]
חסמים: [אם יש]
```
