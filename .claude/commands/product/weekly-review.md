# ישיבה שבועית — מחלקת מוצר

אתה מנחה את הסקירה השבועית של מחלקת המוצר (כל ראשון).

## 1. מדדי שבוע

```
🎯 מוצר — שבועי [תאריך]
• NPS: [X] (יעד: 70)
• Features completed: [N]
• Features in progress: [N]
• Bug reports: [N]
• User research insights: [N]
```

## 2. Feature Review

CPO + ביקורת:
- מה הושלם השבוע ונשלח לפרודקשן?
- מה נחסם ולמה?
- מה הוסף ל-backlog?

## 3. User Feedback

Research Agent מציג:
- מה 3 top feedback מהשבוע?
- מה שינוי ב-NPS אם נמדד?
- מה feature request שחוזר הכי הרבה?

## 4. Roadmap Check

```
OKR CPO: NPS 70+
━━━━━━━━━━━━━━━
NPS נוכחי: [X]
Gap: [70-X]
תכנית: [מה features יעלו NPS?]
```

## 5. עדיפויות שבוע הבא

## דוח CPO ל-CEO
```
🎯 PRODUCT WEEKLY — [תאריך]
NPS: [X]/70
Shipped: [features]
Blocked: [X] → צריך [Y]
Next Sprint: [מה בונים]
```
