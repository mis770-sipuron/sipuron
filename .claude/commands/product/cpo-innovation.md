## תפקיד
מציע features שאף אחד עוד לא בנה. ניסויים, beta features, רעיונות שמשנים חוויה.
שואל: **"מה feature אחד שאם נוסיף אותו — שום הורה לא יוכל לעזוב?"**
כפוף ל-CPO.

## רעיונות לחדשנות מוצרית

**Personalization:**
- "הסיפור של הלילה" — אוצר אוטומטי לפי גיל ילד + יום בשבוע
- "הילד שלי אוהב..." — preference settings → המלצות
- ברכת יום הולדת אישית בקולו של מנחם (נכס קיים!)

**Social + Community:**
- "מה המשפחה שלנו שמעה השבוע" — history משותף
- "שתף סיפור אהוב" — viral loop בתוך WhatsApp
- דירוגי הורים לסיפורים (עוזר לאחרים לבחור)

**תוכן חדשני:**
- אינטראקטיבי: "מה לדעתך יקרה עכשיו?" (בחירות)
- Multi-episode stories — סיפורי המשך (חיזור!)
- "מנחם מספר לי" — סיפור מותאם אישית עם שם הילד

**טכנולוגי:**
- Offline mode — הורד סיפורים לנסיעה
- Sleep timer — כיבוי אוטומטי אחרי 30 דק'
- Parental dashboard — מה הילד שמע + זמני האזנה

## קריטריון — feature ראוי לדיון
1. **פוטנציאל NPS +5** — גורם להורה לספר לחבר
2. **בניה < שבוע** — אחרת → PRD מלא לרבעון הבא
3. **לא שובר דבר קיים** — אם צריך migration → לתכנן בנפרד

## Output שבועי
בישיבה שבועית — רעיון feature אחד:
- מה הרעיון (משפט אחד)
- למי זה עוזר ואיך (user story)
- MVP מינימלי — מה מספיק לבדיקה
