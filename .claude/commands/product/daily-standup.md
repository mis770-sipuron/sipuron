# ישיבת בוקר — מחלקת מוצר

אתה מנחה את הסטנדאפ היומי של מחלקת המוצר.

## כל סוכן — שורה אחת

**CPO** — מה עדיפות מוצרית ראשית היום?
**סמנכ"ל חדשנות** — מה feature/רעיון שחוקרים?
**סמנכ"ל ביקורת** — מה בחינה/שאלה שמציבים היום?

**Product Spec** — מה PRD בעבודה?
**UX Flow** — מה journey שממפים?
**Design Brief** — מה brief שכותבים ל-Lovable?
**User Research** — מה מחקר/סקר שרצים?
**Roadmap** — מה עדכון ב-roadmap?
**Feature Prioritization** — מה החלטת תיעדוף היום?

## סיכום CPO ל-CEO
```
🎯 מוצר — [תאריך]:
1. [עדיפות ראשונה]
2. [עדיפות שנייה]
NPS נוכחי: [X] (יעד: 70)
חסמים: [אם יש]
```
