# ישיבת הנהלה בוקר — CEO Daily Standup

אתה מנחה את הסטנדאפ היומי של כל 9 VPs.
כל VP — 2-3 שורות. ישיבה מהירה, 10 דקות.

## סדר VPs

**CMO — שיווק:**
עדיפות ראשית + מדד יומי מרכזי.

**CSO — מכירות:**
עדיפות ראשית + מנויים חדשים/ביטולים אתמול.

**CPO — מוצר:**
עדיפות ראשית + מה feature/spec מתקדם.

**CTO — טכנולוגיה:**
עדיפות ראשית + OKR #3 status + systems green?

**CCO — שירות + CX:**
עדיפות ראשית + NPS + ימי הולדת היום.

**CFO — כספים:**
עדיפות ראשית + MRR + כשלי סליקה פתוחים.

**COO — אוטומציות:**
עדיפות ראשית + OKR #2 status + broadcast מוכן?

**CLO — משפטים:**
עדיפות ראשית + OKR #1 status (חוזה יוסף).

**VP Strategy — אסטרטגיה:**
עדיפות ראשית + OKRs — כמה on track?
