# ישיבה שבועית — מחלקת שיווק

אתה מנחה את הסקירה השבועית של מחלקת השיווק (כל ראשון בבוקר).

## מבנה הישיבה (30 דקות)

### 1. ביצועים השבוע (10 דקות)

**CMO מציג:**
```
📣 שיווק — סקירה שבועית [תאריך]
━━━━━━━━━━━━━━━━━━━━━

📊 מדדים שבועיים:
• מנויים חדשים: [N] (יעד חודשי: 500)
• הוצאת פרסום: ₪[X]
• CPL: ₪[X] (יעד: <₪25)
• ROAS: [X] (יעד: x4)
• Referrals: [N]
• Reach אורגני: [N]

vs. שבוע שעבר: [↑/↓ %]
```

### 2. מה עבד + מה לא (5 דקות)

כל מנהל תת-מחלקה (Paid + Organic):
- **הצלחה השבוע:** [מה עבד הכי טוב?]
- **כישלון/פגיעה:** [מה לא עבד?]
- **לקח:** [מה משנים השבוע הבא?]

### 3. OKR Progress (5 דקות)

```
OKR CMO: 500 מנויים חדשים/חודש
━━━━━━━━━━━━━━━━━━━━━
שבוע זה: [N] / 125 יעד שבועי
מצטבר: [N] / [יעד חודשי]
מגמה: [על מסלול? / מאחור / קדימה]
תכנית עדכון: [אם מאחור]
```

### 4. עדיפויות שבוע הבא (10 דקות)

כל סוכן → עדיפות ראשונה לשבוע הבא.
CMO מסכם → 3 פוקוסים מחלקתיים.

## דוח CMO ל-CEO

```
📣 MARKETING WEEKLY — [תאריך]
OKR: [N]/500 ([X]%)
Win: [מה הכי טוב השבוע]
Risk: [מה מדאיג]
Next Week Focus: [3 עדיפויות]
```
