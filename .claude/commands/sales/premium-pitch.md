```

בחודש הראשון שמעתם כבר X סיפורים.
שנה שלמה = [X×12] סיפורים ב-₪197 בלבד.
זה ₪16.4 בחודש — במקום ₪47.
מעוניינים? ← [לינק]"
```

**מנוי ותיק (חודש 3+):**
```
"[שם], 3 חודשים יחד! 🙏
[X] סיפורים ב-[Y] ימים — הילד אוהב.

```
