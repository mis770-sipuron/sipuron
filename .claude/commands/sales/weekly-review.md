```

• ביטולים: [N]

• Churn rate: [X]%
• Re-engaged: [N]
• Win-back: [N]
• Upsells (שדרוג): [N]

## 2. Funnel Analysis

**CSO + ביקורת מציגים:**
- איפה נשרים הכי הרבה?
- מה conversion כל שלב (Clarity/Trust/Experience/Subscribe)?
- מה שינוי מהשבוע שעבר?

## 3. OKR Progress

```
OKR CSO: Churn < 3%/חודש
━━━━━━━━━
מדד השבוע: [X]%

- מה ההשקה / קמפיין הבא?
- מה focus retention?
- מי בסיכון גבוה?
